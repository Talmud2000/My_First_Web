소개

경기과학기술대학교 재학중인 컴퓨터모바일융합과 김병록입니다.

공부한 과목

HTML, CSS, JavaScript

웹 페이지의 구조와 내용은 - HTML 태그로 작성되어진다.


웹 페이지의 모양 - CSS(Cascading Style Sheet)로 작성한다.

*cascading : 연속화


웹 페이지의 행동 및 응용 프로그램 - JavaScript 로 작성한다.


해당 웹 페이지에 들어간 태그들의 설명

HTML 태그들
<nav> 네비게이션 효과로 찾고자 하는 목록을 바로 찾을수 있습니다.
<div> 는 <p> 와 함께 블록을 구성하기 위해 가장 많이 사용됩니다. <p> 는 문단을 만들기 위해,
<div> 는 여러 HTML 태그들을 블록으로 묶는 컨테이너로 이용됩니다.
<ul> 은 순서 없는 리스트(Unordered list) 입니다. 그리고 <li> 와 함께 작성 되는데 이는 목록을 만드는 태그 입니다.
<section> 태그는 콘텐츠 영역을 나타냅니다.

CSS 태그들
우선 style.css 라는 파일을 만들어 <href> 로 현재 웹 페이지에 연결 시켜줍니다.
해당 파일에는 HTML태그들을 디자인을 지정해줄 코드가 적혀있습니다.
